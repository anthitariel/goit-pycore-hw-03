# Домашнє завдання з теми: "Корисні модулі Python" (ДЗ №3)

Цей репозиторій містить розв'язання чотирьох завдань, спрямованих на роботу з датами, генерацію випадкових чисел та обробку текстових даних за допомогою регулярних виразів.

## Зміст завдань

1. **Завдання 1 (`task_1.py`)**: Функція `get_days_from_today(date_str)`, яка обчислює кількість днів між заданою датою (у форматі 'РРРР-ММ-ДД') та поточною датою.
2. **Завдання 2 (`task_2.py`)**: Функція `get_numbers_ticket(min, max, quantity)`, що генерує набір унікальних випадкових чисел для лотереї, відсортованих за зростанням.
3. **Завдання 3 (`task_3.py`)**: Функція `normalize_phone(phone_number)`, яка очищує телефонні номери від зайвих символів та приводить їх до міжнародного формату `+380...`.
4. **Завдання 4 (`task_4.py`)**: Функція `get_upcoming_birthdays(users)`, яка визначає список колег, кого потрібно привітати з днем народження протягом наступних 7 днів (з урахуванням перенесення привітання з вихідних на наступний понеділок).
